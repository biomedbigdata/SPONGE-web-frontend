Download ceRNA interactions

The competing endogenous RNA interactions shown in this database have been computed using the SPONGE bioconductor R package (https://doi.org/doi:10.18129/B9.bioc.SPONGE) with matched gene and miRNA expression data from The Cancer Genome Atlas as reprocessed by the TOIL project (https://dx.doi.org/10.1038%2Fnbt.3772).

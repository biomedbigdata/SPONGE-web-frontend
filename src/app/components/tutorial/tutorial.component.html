What is SPONG-web good for?

MicroRNAs are important small non-coding transcriptional regulators involved in many biological processes and human diseases. It is assumed that at least half of the genes in the human genome are influenced by miRNA regulation. Since the number of transcripts with miRNA binding sites far exceeds the number of different miRNAs, these are in competition, creating a complex competing endogenous RNA (ceRNA) interaction network. We applied SPONGE, a method for the fast construction of genome-wide ceRNA interaction network using sparse partial correlation, to infer ceRNA regulatory networks for 23 cancer types in The Cancer Genome Atlas. To make these networks accessible for biomedical research, we created SPONGE-web, a web resource featuring an application programming interface that allows software developers and data analysts to query and filter ceRNA networks efficiently. As an application case and to make this rich resource easily accessible, we further developed a web interface that allows users to visualize ceRNA networks interactively. Access to the web interface or the API requires no login or registration. SPONGE-web enables its users to identify potent ceRNA regulators via network centrality analysis and to assess their potential as a cancer biomarker through survival analysis. In summary, SPONGE-web is a novel, feature-rich web resource supporting the community in studying ceRNA regulation.

How to use the search bar on home

You can enter a gene symbol, ENSG number or miRNA to search in the SPONGE database for all corresponding interactions.
On the next side you will find tables of the found cancer sets. You can search those tables for other genes/miRNAs or create a network from those table entries by clicking the "Show as Network" Button. You can also select some entries and export only those to the network.

Search Features:

Function Effect
Search Box Search for a ENSG number or Gene Symbol. Check the box to only obtain significant results
Result Boxes Lists the cancer datasets, which contain interactions with the searched value. The tables visualize values such as mscor, pvalue, correlation and ids
Options Button Set a range for the Mscore or P-value. You dont have to submit your entry, it will be applied automatically
Show as Network Button Exports selected table entries or default ones to create a interaction network.

Browse functionalities explained

On the browse site you can create different interaction networks and investigate which genes/miRNAs affect each other. The node sizes and edge thicknes is corresponding to the Mscore and node degree. When loading a new cancer set, also a expression heatmap will be produced.
Network-Features:

Function Effect
Title Banner Select Select one of the cancer set, which should be used for a network
Click on a Node Survival Analysis (Kaplan-Meier Plot) will be displaied for the Node
Hover over Node/Edge Values such as Mscor, pvalue, betweenes, correlation and gene ids will be displayed in a side table
Search under network Select a ENSG number or interaction of interest (number) and it will be colored and focused on in the network
Save as You can save & download the plot as .jpg, .png or .svg.
Reset camera Reset the viewpoint of the network after zooming or focusing on a node
Force atlas 2/ start layout Start a layout for a better visualization of the network
Reset Colors Deselect clicked nodes
